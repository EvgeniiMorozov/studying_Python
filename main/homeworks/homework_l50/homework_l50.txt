Создать базу данных MongoDB для учета товаров в магазине.

Для этого потребуется:

1. Создать базу данных storage
2. Создать в ней коллекцию products
3. Наполнить коллекцию документами по своему усмотрению.
Например, там может быть:
3.1. Имя продукта
3.2. Количество
3.3. Дата завоза
3.4. Цена
3.5. Прочие параметры

В личный кабинет выложить скрипт в виде отдельного файла, либо txt.
